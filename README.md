# GameCore — 7TogkSMP

Plugin core server Minecraft **7TogkSMP** berbasis **Nukkit MOT**.

Repo ini dibuat publik buat dokumentasi karya gw di server 7TogkSMP. Gabut sih, tapi ya biar keabadikan. :V

---

## Fitur

- **Economy API** — sistem ekonomi terpusat
- **Placeholder API** — placeholder custom buat scoreboard, chat, dll
- **Voting System** — reward otomatis buat yang udah vote
- **Job System** — pilih pekerjaan, dapet duit dari aktivitas in-game
- **Quest System** — misi harian/mingguan dengan reward
- **Fishing System** — sistem pancing custom dengan loot table sendiri
- Dan masih banyak lagi...

---

## Platform

- Nukkit MOT
- Minecraft Bedrock Edition

---

## Catatan

Plugin ini dibuat spesifik buat server 7TogkSMP, jadi kalau mau dipake di server lain mungkin perlu ada penyesuaian.
